# DeepE

## Run using docker (linux)

### Prerequisites
- Have **_docker_** installed on your PC, you can follow this [guide](https://docs.docker.com/get-docker/).
- Have **_docker-compose_** installed on your PC, you can follow this [guide](https://docs.docker.com/compose/install/).

### Run
Run using the script run.sh

```shell
bash run.sh
```

## Run locally

### Prerequisites
- Have _**nodejs**_ installed on your PC, you can follow this [guide](https://nodejs.org/en/download/).
- Have **_python_** installed on your PC, you can follow this [guide](https://www.python.org/downloads/).

### Run

### Run the following python commands
```shell
npm install
```
```shell
npm install -D tailwindcss
```
```shell
npx tailwindcss -i ./static/src/input.css -o ./static/dist/css/output.css
```
```shell
python3 -m venv venv
```
```shell
source venv/bin/activate
```
```shell
pip install -r requirements.txt
```
```shell
python3 -m flask run --host=0.0.0.0
```
